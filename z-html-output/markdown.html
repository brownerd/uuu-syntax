# Statical
This is a static site starter that uses:
- Jade
- Gulp
- Stylus
- PostCSS
- Browserify/Watchify/Babelify
---
## Requirements
You must **have** Node 4+ *installed*. [ Node](https://nodejs.org/en/ )
---
## Gulp tasks
`gulp ghp`
This is used to copy all the files in the `dist` directory to the `gh-pages` branch for
displaying the site.
